BBC BASIC for BIT90
===================

In this folder are the sources for a customized version of BBC BASIC for the 
BIT90 Home computer.

Changelog:
----------
V1.1: 
- improved line editor (e.g. cursor movement, escape key)
- keywords entered in lowercase are converted to uppercase (variables are not)
- restructured the modules, basic.asm is moved to common folder
- bugfix CLG command
V1.0: 
- initial release
